# Customer-Churn-Analysis-Dashboard
An interactive dashboard to analyze customer churn patterns using Tableau by using KPI's, filters, calculated fields, and interactive visualizations for business insights.

Analyzed customer churn behavior in a telecom dataset to identify high-risk segments and recommend data-driven strategies to improve customer retention and revenue.

Built a multi-layer interactive dashboard to analyze customer churn patterns using Tableau
Developed KPIs such as churn rate, customer segmentation, and revenue metrics
Identified key churn drivers including contract type, tenure, and monthly charges
Implemented filters, calculated fields, and interactive visualizations for business insights

#
Worked with 7,000+ customer records including demographics, subscription details, and billing data
Performed data cleaning and preprocessing:
Handled missing values in TotalCharges
Converted data types for accurate analysis
Created derived features:
Churn Flag (binary classification)
Tenure Segmentation (customer lifecycle grouping)
High-Risk Customer classification

#
KPI Development
Defined business metrics:
Churn Rate (%)
Customer Retention Rate
Average Revenue per User (ARPU)
Customer Lifetime Value (approx using tenure × monthly charges)
Average Tenure
